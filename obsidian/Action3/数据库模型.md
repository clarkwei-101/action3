# 数据库模型 (Prisma Schema)

## 核心数据模型

### UserProgress
用户进度追踪
```prisma
model UserProgress {
  id            String   @id @default(cuid())
  totalXP       Int      @default(0)
  level         Int      @default(1)
  currentStreak Int      @default(0)
  longestStreak Int      @default(0)
  lastActiveAt  DateTime?
}
```

### Goal
学习目标
```prisma
model Goal {
  id            String   @id @default(cuid())
  title         String
  description   String?
  style         String   @default("guided")
  progress      Float    @default(0)
  targetDate    DateTime?
  completedAt   DateTime?
  createdAt     DateTime @default(now())
  milestones    Milestone[]
  tasks         DailyTask[]
}
```

### Milestone
目标里程碑
```prisma
model Milestone {
  id        String   @id @default(cuid())
  goalId    String
  title     String
  completed Boolean  @default(false)
  goal      Goal     @relation(...)
}
```

### DailyTask
每日任务
```prisma
model DailyTask {
  id        String    @id @default(cuid())
  goalId    String
  title     String
  completed Boolean   @default(false)
  completedAt DateTime?
  goal      Goal      @relation(...)
}
```

## 成就系统

### Achievement
```prisma
model Achievement {
  id            String    @id @default(cuid())
  key           String    @unique
  name          String
  description   String
  icon          String
  category      String    // starter|streak|milestone|style|master
  triggerType   String
  triggerConfig String
  xpReward      Int
  levelRequired Int       @default(1)
  unlockedAt    DateTime?
}
```

## 技能系统

### SkillNode
技能树节点
```prisma
model SkillNode {
  id          String @id @default(cuid())
  title       String
  description String
  category    String  // frontend|backend|ai
  difficulty  String
  x           Float
  y           Float
}
```

### SkillEdge
技能关系
```prisma
model SkillEdge {
  id           String @id @default(cuid())
  sourceNodeId String
  targetNodeId String
}
```

### LearningPath
学习路径
```prisma
model LearningPath {
  id          String @id @default(cuid())
  name        String
  description String
  nodes       LearningPathNode[]
}
```

### SkillMastery
技能掌握度
```prisma
model SkillMastery {
  id        String @id @default(cuid())
  skillNodeId String
  mastery   Int     @default(0)  // 0-100
}
```

## 等级计算公式
```
升级所需 XP = level * level * 100
例: Lv2 = 400XP, Lv3 = 900XP, Lv5 = 2500XP
```
