# Action3 项目概览

## 简介

Action3 是一个基于 Big-AGI 构建的学习目标管理平台，集成 AI 辅助、目标追踪、技能树、语音播报、课堂学习等功能。

## 技术栈

- **前端**: Next.js 15, React 18, TypeScript
- **状态管理**: Zustand + IndexedDB
- **API**: tRPC (Edge + Cloud 分离)
- **AI 服务**: DeepSeek API
- **数据库**: SQLite (via Prisma)
- **可视化**: React Flow (技能树)
- **样式**: CSS 变量 + 内联样式

## 核心功能

1. **目标管理** - 创建/编辑/删除学习目标，支持5种学习风格
2. **AI 辅助** - 自动分解目标为里程碑和任务
3. **技能树** - React Flow 可视化技能关系图
4. **AI 课堂** - 4角色多智能体讨论
5. **成就系统** - 27个成就，分5类
6. **语音播报** - 多语言 TTS，支持 zh/en/ja/ko
7. **日历集成** - iCal 导入，AI 分析空闲时间
8. **推送通知** - Web Push API
