# 组件清单

## 页面组件

### `/action3/goals` - 目标管理
- 目标列表展示（卡片布局）
- 目标创建表单（标题、描述、风格、截止日期）
- 目标详情弹窗（里程碑 + 任务）
- 任务完成/创建
- AI 分析按钮

### `/action3/recommend` - 推荐
- AI 推荐技能
- AI 推荐学习路径
- AI 推荐任务
- 推荐详情展示

### `/action3/skilltree` - 技能树
- React Flow 可视化
- 节点点击详情
- 掌握度进度条
- 学习路径高亮
- 缩放/拖拽控制

### `/action3/classroom` - AI 课堂
- 4角色流式对话（Teacher + 3 Students）
- 消息历史
- 输入框
- 加载状态

### `/action3/calendar` - 日程管理
- 月历视图
- 日期选择
- 事件列表
- 添加事件弹窗
- ICS 导入
- 空闲时间 AI 分析

### `/action3/browser` - 浏览器
- Playwright 集成
- 截图表述
- URL 导航
- 命令执行（截图、刷新、滚动、内容提取）
- 日志面板
- 快捷操作

### `/action3/voice` - 语音播报
- Web Speech API TTS
- 多语言支持（zh/en/ja/ko）
- 播报风格选择
- 语速/音调控制
- AI 对话助手
- 今日任务展示

### `/action3/achievements` - 成就系统
- 27 个成就卡片（5 分类）
- 毛玻璃 UI 效果
- 已解锁/未解锁状态
- 详情弹窗（解锁时间、XP 奖励）
- 用户等级卡
- 统计信息

### `/action3/assessment` - 目标评估
- 目标选择
- 评估类型（quiz/project_review/certification）
- AI 生成题目
- 答题界面
- 评分展示
- 反馈

### `/action3/settings` - 设置
- 用户信息（用户名）
- 默认学习风格选择
- 早间/晚间提醒
- Web Push 通知开关
- 主题（仅深色）
- 保存按钮

## 共享组件

### Action3Layout
统一布局，包含：
- 72px 侧边导航栏
- CSS 变量注入（--bg-*, --accent-*, --text-*, --border-*）
- 动画 orb 背景
- 成就解锁通知（banner + overlay）
- Toast 通知容器

### BackButton
返回按钮组件，支持返回上一页或指定路径。

## API Hooks

| Hook | 描述 |
|-------|------|
| `useAction3Goals` | 获取目标列表 |
| `useAction3Tasks` | 获取任务列表 |
| `useAction3Achievements` | 获取成就列表 |
| `useAction3Progress` | 获取用户进度 |
| `useAction3Calendar` | 获取日历事件 |
| `useAction3TasksToday` | 获取今日任务 |

## 服务层

| 服务 | 描述 |
|------|------|
| `action3-ai.service` | 目标分析、分解 |
| `action3-recommendation.service` | 技能/路径推荐 |
| `action3-classroom.service` | 多角色 AI 课堂 |
| `action3-assessment.service` | 评估题目生成、评分 |
| `action3-research.service` | Web 调研 |
| `action3-voice-assistant.service` | 问候语、对话 |
| `action3-ical-import.service` | ICS 解析 |
